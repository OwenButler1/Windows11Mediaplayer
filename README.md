# Windows11Mediaplayer

Minimal beta-ready Windows 11 media player shell built with Electron.

## Prerequisites

- Windows 11
- Node.js 20+
- npm 10+

## Run locally

```bash
npm install
npm start
```

## Beta validation flow

1. Launch the app with `npm start`.
2. Click **Open Local Media**.
3. Pick any local audio/video file.
4. Confirm the load status shows the selected path and the player appears.
